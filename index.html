Mine the Connections among Stocks

It has been widely accepted that Domino Effect of Board Linkage exist in security market. Intuitively thinking, prices of certain stocks tend to go up and down together, either because they are in the same industry or because they are from the same region. This is especially meaningful for portfolio management.

Here we provide an interactive demo to demonstrate how such data mining techniques help improve human's understanding of the stock market. Given the stocks (specified by symbols) in the textbox below, we are able to identify their structure from the price correlations in 2012, and visualize such structure with a graph.

We've already put an example set in the textbox. Click "Run!" to build the visualization. You can also modify the symbols (should be indexed by S&P 500) by yourself and check the results. If the default stocks are used, it's obvious that the financial stocks and IT stocks are separated into two clusters within our visualization.

This demo involves python for stock data crawling, javascript for data analysis and visualization.
